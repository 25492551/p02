## 3. 수학적 배경

### 3.1 벡터장의 의미

벡터장 V(x, y)는 각 점에서의 흐름 방향과 속도를 나타냅니다:

```
V(x, y) = -∇E(x, y)
```

여기서:
- ∇E: 에너지 기울기
- 음의 부호: 에너지 감소 방향

### 3.2 싱크(Sink)의 개념

- **싱크**: 주변 흐름을 빨아들이는 점
- **영점**: 에너지 최소점 = 싱크
- **유역(Basin)**: 싱크로 수렴하는 영역

### 3.3 회전성(Vorticity)

벡터장의 회전:

```
ω = ∇ × V
```

영점 주변에서 회전 패턴이 나타날 수 있음.

1. **영점 근처**:

   - 속도가 빠름 (빨간색)

   - 대부분의 흐름이 임계선을 따라 이동

3. **영점 간 영역**:
