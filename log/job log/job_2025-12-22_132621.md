#### 새로운 폴더 구조
- `fig/`: 모든 시각화 그림 파일 저장 (새로 생성)
- `data/`: 논문 제출 관련 문서 및 데이터 파일
- `script/`: Python 스크립트 파일
- `report/`: 분석 리포트 파일
- `log/`: 작업 로그 및 채팅 로그

### 2. .gitignore 업데이트

#### 변경 내용
- **추가된 항목**: `data/` 폴더를 Git 추적에서 제외
- **이유**: 
  - 논문 제출 관련 문서 (PDF, LaTeX 컴파일 파일 등)는 Git에 포함하지 않음
  - 저장소 크기 최적화
  - 개인 정보가 포함된 제출 문서 보호

#### .gitignore 현재 상태
```
# Python 관련
__pycache__/
*.py[cod]
*$py.class
*.so
.Python

# Virtual Environment
venv/
env/
ENV/
.venv

# IDE
.vscode/
.idea/
*.swp
*.swo
*~

# OS
.DS_Store
Thumbs.db

# Logs (optional)
# log/

# Journal (제출 문서)
data/
```

### 3. 파일 구조 개선 효과

#### 장점
1. **명확한 분리**: 그림 파일과 문서 파일의 명확한 구분
2. **Git 관리 최적화**: 제출 문서는 Git에서 제외하여 저장소 크기 감소
3. **프로젝트 구조 명확화**: 각 폴더의 역할이 더 명확해짐
4. **유지보수 용이성**: 파일 위치가 논리적으로 정리됨

#### 영향받는 파일
- **스크립트 파일**: 그림 저장 경로가 `data/`에서 `fig/`로 변경 필요 (향후 업데이트 필요)
- **LaTeX 파일**: 그림 참조 경로 업데이트 필요 (향후 업데이트 필요)
